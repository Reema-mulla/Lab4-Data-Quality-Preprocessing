# Lab4-Data-Quality-Preprocessing
**Student:** Reema AlMulla  

## Overview
This assignment focuses on assessing data quality and applying preprocessing techniques to prepare the dataset for analysis and machine learning tasks. The Chocolate Sales dataset was used to demonstrate different preprocessing methods.

## Dataset
The dataset used in this assignment is **Chocolate_Sales.csv**, which contains sales records including:
- Sales Person
- Country
- Product
- Date
- Amount
- Boxes Shipped
